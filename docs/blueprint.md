# **App Name**: LedgerStream

## Core Features:

- Homepage: A modern and intuitive homepage for the SaaS accounting web app.
- Authentication System: Secure user login and sign-up with email/password and Google authentication.
- User Role Management: Define and assign different roles to users, controlling access to features and data within the application.

## Style Guidelines:

- Primary color: A deep, professional teal (#176868) representing clarity and financial stability.
- Background color: A very light, desaturated teal (#E8F2F2) that provides a clean, bright, and spacious feel, almost white.
- Accent color: A soft, cool green-blue (#73B89A) for subtle highlights and complementary elements, enhancing readability without distracting.
- All text: 'Inter' (sans-serif) for its clean, modern, and highly legible appearance, suitable for a professional business application.
- Utilize a suite of professional, minimalist line-art icons that consistently reflect financial and business concepts.
- Clean, card-based layout on the dashboard with ample whitespace to ensure clear distinction and focus on data points.
- Subtle and swift transition animations for dashboard widgets and navigation elements, providing a responsive and fluid user experience.
